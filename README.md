# 🎥 Telegram Video Compressor Bot

A powerful and super-fast Telegram bot that compresses videos.

## 🛠 Features

- ✅ Compress any video up to 2GB
- ⚡ Super-fast download, compress, and upload
- 📊 Live progress bar (with percentage & file size)
- 🏷️ Adds suffix in filename
- 📦 Streamable output
- 🌐 Works in Groups and Private Chat
- 🧩 Flask integrated for uptime (Koyeb, Heroku, Render etc.)
- 👥 Group compatible

## 🚀 Deployment

### 📦 Requirements

- Python 3.9+
- FFMPEG installed
- Telegram Bot Token
- Pyrogram & TgCrypto

### 🧪 Installation

```bash
git clone https://github.com/bot-dd/Vid-Compress-bot
cd video-compressor-bot
